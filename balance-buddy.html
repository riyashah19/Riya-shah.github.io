Project Overview

Balance Buddy is a wearable stability-monitoring belt designed to reduce fall risk during exercise and rehabilitation for older adults. With approximately one in three older adults falling every year, and limited staff supervision in group exercise settings, we created a solution that monitors lower-body stability in real-time while enabling staff to supervise multiple users simultaneously.

Key Contributions

- Designed and prototyped the wearable belt system with a rotary angle sensor connected to thigh band for tracking lower-body stability.

- Built real-time monitoring dashboard using HTML and CSS, enabling staff to supervise multiple users at once.

- Developed personalized threshold calibration for each user to distinguish normal movement from instability.

- Implemented emergency alert button (double-press) for user-initiated calls for help.

- Designed a lever on SolidWorks that was attatched to the thigh band, ensuring smooth reaction timings with leg movements and the webpage.
