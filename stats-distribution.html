Statistical Distribution Fitting Tool

Project Overview

I built an interactive Streamlit web application that allows users to fit statistical distributions to their data.

The app provides both automatic fitting using SciPy's distribution methods and manual parameter adjustment via sliders.

Users can enter data manually or upload CSV files, visualize the results, and export fit parameters.

Distributions Implemented

Normal Gamma Weibull Exponential Uniform Logistic Lognormal Beta Chi-square Rayleigh
